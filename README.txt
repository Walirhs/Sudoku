Pour lancer l'installation des bibliothèques SDL sous linux : Se placer dans le répertoire du projet,
et exécuter la commande bash install.sh .
Cela lancera les commandes d'installations des bibliothèques SDL et exécutera le programme.

Si vous souhaitez simplement lancer le projet, exécuter la commande bash install.sh 
et refuser le téléchargement.
